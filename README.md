# Pizza Sales Dashboard

## Project Overview
This project analyzes pizza sales data to identify
revenue patterns, customer behavior, and product performance.

## Tools Used
- Google BigQuery
- Looker Studio
- SQL

## Key KPIs
- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas per Order

## Key Findings
- Fridays and Saturdays have the highest orders
- January and July are the peak sales months
- Classic category contributes the highest sales
- Large size pizzas generate the most revenue
- Thai Chicken Pizza is the top revenue earner
- Brie Carre Pizza is the lowest performer

## Charts Used
- Bar Chart : Orders by day and revenue by size
- Line Chart : Monthly sales trend
- Pie Chart : Sales by category
- Horizontal Bar Chart : Best and worst selling pizzas
